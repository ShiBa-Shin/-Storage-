# Typefaces

## __Comparison__

| Typeface      | Mono | Ligature | Open Source | Free |
| ------------- | :--: | :------: | :---------: | :--: |
| JeBrains Mono |  ✓   |    ✓     |      ✓      |  ✓   |
| IBM Plex Mono |  ✓   |    ✗     |      ✓      |  ✓   |
| MonoLisa      |  ✓   |    ✓     |      ✗      |  ✗   |
| Hack          |  ✓   |    ✗     |      ✓      |  ✓   |

If you like using the ligature feature but hasn't been supported by the font yet, maybe you can find the patched version on [NERD FONTS](https://www.nerdfonts.com/).
